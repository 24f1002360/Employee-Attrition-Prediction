# Employee Attrition Prediction using Machine Learning

## Project Overview

Employee attrition is one of the major challenges faced by organizations as it leads to increased recruitment costs, training expenses, and productivity loss.

This project develops a Machine Learning model to predict whether an employee is likely to leave the organization using the IBM HR Analytics Employee Attrition dataset.

The project includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Building
- Model Evaluation
- Business Insights & HR Recommendations

---

## Dataset

IBM HR Analytics Employee Attrition Dataset

Rows: **1470**

Features: **35**

Target Variable:

- Attrition
  - Yes
  - No
